# Andean Amylase Project

Location of all created code and data associated with the paper "Rapid Adaptive Increase of Amylase Gene Copy Number in Indigenous Andeans ". Files that are in .rds format can be opened in r.

The Quechua_Maya_PhasedChr1_103343000_103960000_no_100_non_AMR_Ancestry_in_locus.vcf.gz contains SNPs for the Quechua and Maya samples in the region between 103343000 and 103960000 for all individuals that do not have 100% non-American ancestry in the locus.

DOI: 10.5281/zenodo.18226081
